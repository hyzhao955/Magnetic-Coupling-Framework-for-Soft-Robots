# Magnetic-Coupling-Framework-for-Soft-Robots
This is a framework for the dynamic simulation of the magnetic soft robots. The framework is based  on SOFA. 


The example program demonstrates the use of an equivalent mapping algorithm for magnetic torque-magnetic nodal force, derived from the paper "A FEM Framework for Simulating Magnetic Soft Continuum 
Robots via Consistent Torque-to-Nodal Force Mapping" submitted to MARSS conference.


The plugin needs to be configured according to the official Sofa instructions.
